⚙️ Overview

I design and operate a virtualized lab environment used to experiment with modern infrastructure platforms, monitoring systems, automation tooling, and IT operations software.

The goal of this lab is to simulate a real-world enterprise infrastructure where systems can be deployed, monitored, automated, and documented.

This GitHub profile serves as a technical portfolio of the systems currently running in my environment, while detailed configuration and deployment guides are documented within the individual repositories.

🖥️ Homelab Environment

The lab is built on Microsoft Hyper-V and consists of multiple virtual machines used to replicate a small enterprise infrastructure.
